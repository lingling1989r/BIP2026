## 🚀 快速开始（5分钟上手）

### 第一步：选择目标社群
```
宝妈社群 → 用场景一物料
IT社群   → 用场景二物料
副业社群 → 用场景三物料
```

### 第二步：准备发布
1. 打开对应场景的文案（在`社群分享物料-三个场景.md`中）
2. 复制完整文案到备忘录
3. 根据实际社群氛围微调语气（参考`使用指南.md`）
4. 准备对应配图（在`social-graphics/`文件夹中）

### 第三步：最佳发布时间
- **宝妈社群**：周一/周三 晚8-9点
- **IT社群**：周二/周四 中午12-1点 或 晚10-11点
- **副业社群**：周末 上午10-11点 或 下午3-4点

### 第四步：互动跟进
- 发布后1小时内回复前5-10条评论
- 准备好私信话术（在`使用指南.md`中查找）
- 设置自动回复或快捷回复

- 先解决问题，再介绍工具
- 强调免费试用，降低决策门槛
- 提供1v1诊断，建立个人连接
